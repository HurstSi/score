
**简要描述：** 

- 用户登录接口

**请求URL：** 
- ` /users/auth `
  
**请求方式：**
- GET

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|code |是  |string | 	小程序登录时获取的 code  |


 **返回示例**

> 失败（未注册）
``` 
{
	"mes": "null",
	"data": ""
}
```
> 成功
```
{
    "mes": "",
    "data": {
        "stuNum": "学号",
        "name": "姓名",
        "token": "成功之后的token",
        "class": "讲台名称"
    }
}
```
