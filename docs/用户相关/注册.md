
**简要描述：** 

- 用户注册接口

**请求URL：** 
- ` /users/ `
  
**请求方式：**
- POST

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|code |是  |string | 	小程序登录时获取的 code  |
|stuNum|是|string|学号|
|name|是|string|姓名|
|class|是|int|讲台id|

 **返回示例**

``` 
{
    "mes": "",
    "data": {
        "stuNum": "学号",
        "name": "姓名",
        "token": "token",
        "class": "讲台名称"
    }
}
```

> 注册之后自动登录
