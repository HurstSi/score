
**简要描述：** 

- 获取项目所有评分
> 需要鉴权（管理员）

**请求URL：** 
- ` /score/item `
  
**请求方式：**
- GET

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|itemId |是  |int |项目id|

 **返回示例**

``` 
{
    "mes": "",
    "data": [
        {
            "name": "用户名称",
            "stuNum": "用户学号",
            "content": "用户评分"
        },
    ]
}
```
