### 用户表：
- id: 自增主键
- openid: 微信唯一识别码
- stuNum: 学号
- name: 姓名
- classId: 班级外键

### 项目表：
- id: 自增主键
- name：项目名称
- logo：项目logo
- info：项目信息
- classId: 班级外键

### 评分表：
- id: 自增主键
- userId: 用户外键
- itemId: 项目外键
- content：成绩

### token表：
- id: 自增主键
- userId: 用户外键
- content: token内容
- createTime: 创建时间

### 班级表
- id: 自增主键
- name: 班级名称
